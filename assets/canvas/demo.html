📖 Jak to funguje

Nový systém je rozdělen do několika modulů:

  • CanvasManager - Správa canvasu a rendering loop
  • FoodSystem - Systém částic s object poolingem
  • PerformanceMonitor - Monitoring FPS a adaptivní kvalita
  • MathUtils - Optimalizované matematické funkce
  • Vrstvy: WaterGradient, LightRays

Klikněte kamkoliv na stránku pro vytvoření částic jídla!

🚀 Integrace

Pro použití v existujícím projektu:

<script type="module">
    import { createCanvasBackground } from './canvas/index.js';
    
    const manager = createCanvasBackground({
        zIndex: 0,
        showStats: true,
        targetFPS: 45
    });
    
    manager.start();
</script>
